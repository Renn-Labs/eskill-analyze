# World-Class Signals

World-class is NOT "really good." It means:

| Signal | What It Looks Like |
|-|-|
| **Delight** | Surprises in a good way. Users smile. |
| **Conversation** | They tell friends unprompted. |
| **Invisible** | Works so seamlessly they don't notice effort. |
| **Loss Aversion** | They'd lose something real by leaving. |
| **Extensible** | Third-parties can build on it. |

**Test**: Would a user miss this if we removed it tomorrow?

## World-Class Criteria Output

3-5 specific, measurable signals that prove we're there. Not aspirational platitudes — concrete metrics or behaviors that can be observed and verified.
